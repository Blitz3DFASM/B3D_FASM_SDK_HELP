cinvoke bbLoadBrush, file, brush_flags=1, u_scale=1.0, v_scale=1.0

Parameters

file (dword addres) - adress to start of string (ended with 0 byte), witch contains file name or path to file
brush_flags (dword integer) - flags can be added to combine effects: 1: Color 2: Alpha 4: Masked 8: Mipmapped 16: Clamp U 32: Clamp V 64: Spherical reflection map (default value is 1)
u_scale (dword float) (default value is 1.0)
v_scale (dword float) (default value is 1.0)

Return

BBBrush
