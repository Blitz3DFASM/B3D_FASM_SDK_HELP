cinvoke bbRect, x, y, width, height, solid=1

Parameters

x (dword integer) - x position of rect
y (dword integer) - y position of rect
width (dword integer) - width of rect
height (dword integer) - height of rect
solid (dword integer) - 1 or BBTrue to create solid object, 0 or BBFalse to create unfilled object (default value is 1)

Return

Nothing
